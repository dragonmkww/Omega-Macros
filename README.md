# ZTweaks-Free-Macros
The ZTweaks keyboard macro is extremely overpriced and I don't really think anyone should give them money. I managed to copy the actual AHK scripts for the macros with my settings, but you can edit them if you know what you're doing.

The devs obviously didn't want anyone to do this, so they made the script files only generate while the macro is running. However, it was really easy to just copy them while the macros were running and paste them elsewhere.

To use the macros, just click on whichever file you want to use and it will automatically turn on the macro. You can turn it off by going to the tray and closing it.

To change the keybinds, right click on the macro you want to edit and open it in VSCode or notepad or anything like that. Then just find the global variables section and change them to whatever your settings are in fort.

You can also change the timing and even add functions to this macros this way.

And no, this isn't illegal distribution. By a stroke of genius, the ZTweaks owners licensed all their products under the Apache License, which allows for free distribution and modification as long as I say that these files are indeed licensed. The ZTweaks owners can't sue me or take down this repo or anything.

Enjoy!
